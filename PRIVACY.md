# Privacy Policy

## Civic Sentiment Dashboard

This application collects and analyzes public Twitter data for sentiment analysis. 

### Data Collection
- We only access publicly available tweets
- No personal user data is stored
- All analysis is anonymous and aggregated

### Data Usage
- Sentiment analysis for civic topics
- Public opinion research
- No commercial use of data

### Contact
For privacy concerns, open an issue at: https://github.com/fallofpheonix/civic-sentiment-dashboard
